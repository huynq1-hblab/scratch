## Nội dung mới trong SOA-C03 (New Topics)

- 🐳 **Containers**: Amazon ECS, Amazon EKS, Docker
- 🏗️ **IaC nâng cao**: AWS CDK, CloudFormation advanced
- 📊 **Observability**: Amazon Managed Prometheus, Amazon Managed Grafana
- 🔐 **Identity**: IAM Identity Center (multi-account)
- 🌐 **Networking**: AWS Network Firewall

## Cách học (Learning Strategy)

1. **Read the MD files**: Mỗi bài học sẽ có giải thích tiếng Việt và thuật ngữ tiếng Anh.
2. **Answer Practice Questions**: Cuối mỗi file sẽ có câu hỏi mô phỏng đề thi.
3. **English Practice**: Tập trung vào các động từ/danh từ kỹ thuật (keywords).
